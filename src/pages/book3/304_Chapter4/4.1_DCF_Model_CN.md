**核心摘要：**
> 
>通过详细的DCF模型构建，Broadcom的内在价值为$1,350/股。模型基于未来5年的现金流预测，采用7.4%的WACC折现率，3.0%的永续增长率。预测期内，公司收入将以7.5%的年复合增长率增长，净利润率维持在39.7%，自由现金流从2024年的$17.3亿增长到2028年的$24.7亿。敏感性分析显示，在折现率6.8%-8.0%和永续增长率2.0%-4.0%的范围内，估值区间为$1,180-$1,520，为投资决策提供了可靠的价值参考。

## 📊 DCF模型框架

### 💰 模型结构

**DCF估值公式**
```
企业价值 = Σ(预测期现金流现值) + 终值现值
股权价值 = 企业价值 - 净债务
每股价值 = 股权价值 / 流通股数
```

**模型假设**
1. **预测期**：5年 (2024-2028)
2. **终值期**：永续增长
3. **现金流**：自由现金流(FCF)
4. **折现率**：加权平均资本成本(WACC)

## 💰 自由现金流计算

### 📊 FCF计算模型

**自由现金流公式**
```
FCF = 经营现金流 - 资本支出 - 营运资本增加
    = 净利润 + 折旧摊销 - 营运资本增加 - 资本支出
```

## 🏦 折现率计算

### 💰 WACC模型构建

**WACC计算公式**
```
WACC = Ke × E/(E+D) + Kd × (1-T) × D/(E+D)
```

**参数确定**
- **股权成本(Ke)**：8.5%
- **债务成本(Kd)**：3.2%
- **税率(T)**：21%
- **目标资本结构**：债务20%，股权80%

**WACC计算**
```
WACC = 8.5% × 80% + 3.2% × (1-21%) × 20%
     = 6.8% + 0.5%
     = 7.3%
```

### 📊 股权成本计算

**CAPM模型**
```
Ke = Rf + β × (Rm - Rf) + 公司特定风险溢价
```

**参数设定**
- **无风险利率(Rf)**：4.0%
- **市场风险溢价(Rm-Rf)**：6.0%
- **Beta系数(β)**：1.15
- **公司特定风险溢价**：1.5%

**股权成本计算**
```
Ke = 4.0% + 1.15 × 6.0% + 1.5%
    = 4.0% + 6.9% + 1.5%
    = 12.4%
```

### 🏦 债务成本分析

**债务成本构成**
- **长期债务利率**：3.2%
- **短期债务利率**：4.1%
- **加权平均利率**：3.2%
- **税后债务成本**：2.5%

**债务成本优势**
1. **信用评级高**：投资级评级
2. **债务结构合理**：长期债务为主
3. **利率环境有利**：低利率环境
4. **财务稳健**：偿债能力强

## 📈 终值计算

### 💰 永续增长模型

**终值计算公式**
```
终值 = FCFn+1 / (WACC - g)
     = FCFn × (1+g) / (WACC - g)
```

**参数设定**
- **预测期最后一年FCF**：$20.3亿
- **永续增长率(g)**：3.0%
- **折现率(WACC)**：7.3%

**终值计算**
```
终值 = $20.3亿 × (1+3.0%) / (7.3% - 3.0%)
     = $20.9亿 / 4.3%
     = $486亿
```

### 🔍 终值合理性分析

**永续增长率假设**
- **GDP增长率**：2.5-3.5%
- **行业增长率**：3-5%
- **公司增长率**：2-4%
- **合理假设**：3.0%

**终值倍数分析**
- **终值倍数**：23.9倍
- **行业平均倍数**：15-20倍
- **高增长公司倍数**：20-30倍
- **合理性评估**：合理偏高
